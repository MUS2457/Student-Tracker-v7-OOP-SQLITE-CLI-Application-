Student tracker v 7 CLI

A simple command-line application written in Python to manage student records.

The program allows users to enter students, add subjects and scores, analyze their performance, and store the data in an SQLite database.

This project was built as a learning exercise to practice Python and basic database integration.

First Time Using SQL

This project is my first time working with SQL and SQLite.
I used it to learn how to:

create tables

use foreign keys

store related data across tables

perform joins to retrieve student information

calculate averages directly from the database

It helped me understand how Python applications interact with databases.

Features

Add students and their subjects

Automatic class analysis

Pass / fail report

Students above and below class average

Best and lowest performing students

Search students in the database

Show top 3 students

Data stored using SQLite
